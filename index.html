Starting the Video System

All audiovisual content is run through and managed by the Crestron DMPS-300-C, located in the media rack to the right of the computer.  This device acts as the bus hub for all audio and video sources that are used for streaming and recording, as well as the projector and booth displays.  This device was designed to run at all times, but to save power we have opted to turn it off after use.  This means the device needs to be manually configured each time we need to use the system until a more permanent program is installed.

Each input and output on the DMPS represents a different device we use during service. Below is a table that can help you remember what each input and output represents.

ID INPUT OUTPUT
1 PC DISP. 1 BOOTH DISPLAY
2 PC DISP. 2 PROJECTOR DISPLAY
3 BOOTH HDMI NOT USED
4 DVD/CD PLAYER NOT USED
5 SOUNDBOARD OUT
Program PC AUDIO SOUNDBOARD IN
AUX 1 NOT USED NOT USED
AUX 2 NOT USED PC REC. IN
